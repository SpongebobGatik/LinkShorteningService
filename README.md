# Сервис сокращения ссылок 🔗

Простой и функциональный сервис для сокращения URL-адресов со встроенной статистикой переходов.

## 🚀 Возможности

- Сокращение длинных URL до короткого 8-символьного кода
- Автоматический переход по короткой ссылке
- Сбор статистики переходов (IP-адрес, время, URL)
- Гибкая система отчётов по различным измерениям
- Встроенная БД

## 🏗️ Архитектура

Система состоит из трёх независимых серверов:

| Сервер | Порт | Технологии | Назначение |
|--------|------|------------|------------|
| Database Server | 6379 | C | Хранение данных (стек, очередь, множество, хеш-таблица) |
| Web Server | 5000 | Python | Сокращение ссылок (основной сайт ip_сайта:5000/form)|
| Statistics Server | 5001 | Python | Сбор и выдача статистики (просмотр через ip_сайта:5001/get_report)|
